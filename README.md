# 📚 Alura Books

Projeto desenvolvido durante o curso da Alura com o objetivo de praticar conceitos de React e componentização.

## 🚀 Tecnologias utilizadas

* React
* JavaScript
* Styled Components
* HTML
* CSS

## 📖 Sobre o projeto

O projeto consiste em uma interface inspirada em uma livraria virtual, contendo:

* Barra de pesquisa de livros
* Seção de últimos lançamentos
* Componentização com React
* Estilização utilizando Styled Components

Este projeto foi desenvolvido apenas para fins de estudo e prática dos conceitos aprendidos no curso da Alura.

## ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone URL_DO_SEU_REPOSITORIO
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npm start
```
